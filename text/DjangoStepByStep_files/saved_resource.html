                        gmasa • a year ago • edited

                        (Posted from Japan, auto-translated by DeepL. Sorry if there are any unnatural phrases.)
                        I tried to improve the "News App" in "Beginners for Django" to a Markdown version. Then I got the following error.
                        Error Description:
                        Error during template rendering.

                        I dealt with it as follows.


                        # config/settings.py
                        ...
                        TEMPLATES = [
                            {
                                ...
                                'OPTIONS': {
                                    'context_processors': [
                                        ... ,
                                    ],
                                    'libraries': { # new
                                         'markdown_extras': 'templatetags.markdown_extras'
                                     }
                                },
                            },
                        ]
                        ...


                        I hope this is helpful to others who have run into the same problem.
